# Otimizando-o-Sistema-Bancario_API---Projeto-5-DIO-

O projeto de desafio consistiu em realizar uma API assíncrona utilizando FastAPI, Alembic, SQLalchemy, autenticação com JWT e Pydantic em Python como base do desafio solicitado nas aulas do curso de Back-end com Python da plataforma DIO, a partir das informações referentes em: https://github.com/digitalinnovationone/trilha-python-dio/tree/main/13%20-%20APIs%20Ass%C3%ADncronas%20com%20FastAPI/desafio

﻿
O projeto em questão, foi realizado com base na continuação do projeto realizado com programação orientada a objeto em Python ( https://github.com/larifq/Otimizando-o-Sistema-Bancario---Projeto-2-DIO- ).
